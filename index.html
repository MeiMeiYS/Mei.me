Yoshi app

Yoshi, a project inspired by Meetup, is as app where gamers can search for gaming groups/party to join, makes friends, and play games together. This is my first solo project at App Academy.

Yoshi was built in one week. During that week, I learned how to build an app from scratch, project planning, web design, time management, and problem solving. I built this app using redux at frontend after learning redux for one week. What I learned the most from this project is researching within documentation, solving problem independently and asking good questions if problems can not be solved on my own.

Plantagram

plantstagram, a project inspired by instagram, is a plants image sharing platform that allows users to publicly share their plants and can be viewed, liked, and commented by other users.

plantstagram is one of my group project at App Academy. There are 4 members in this team, including me. One of the feature I worked on and really enjoy building is the image uploading modal. plantstagram utilized firebase cloud storage for image hosting server, and implemented image preview feature. When a user is creating a new post, they can preview their chosen image on their local computer before sending it to the database.

View Github Repo Live site

Tickety

Tickety is an app where you can manage your work flow by stages and keep track of your tasks. Tickety app is designed & coded by me using React as frontend, Express as backend, and used PostgreSQL/Sequelize to setup database. Tickety app is built in 10 days in 2022 February. It is my capstone project at App Academy.

The most challenging part of building this app for me is implementing the drag and drop task card on the kanban board and dynamically updating the frontend state and backend data with no animation delay.
